# CLI-Contact-Book

## About
A contact book that lives within the terminal that allows to add, delete, search contacts along with displaying the entire contact list. The list also persists between runs allowing for up-to-date contact listing.

## How to run
- clone or download the repo
- do "npm install" within the terminal in order to install all necessary packages.
- run  "node main.js" to start the game.
